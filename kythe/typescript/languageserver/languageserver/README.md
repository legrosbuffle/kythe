## Building

Be sure to run `npm install` from the `languageserver` directory in order to
fetch dependencies

To build the server, run:

```bash
npm run compile
```

or

```bash
npm run watch
```

## Testing

In order to test the server, run:

```bash
npm run test
```

If you wish to run a watch build on tests, run:

```bash
npm run watch_test
```
